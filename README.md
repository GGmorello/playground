# Github Pages

Install:


Run locally:
```
bundle exec jekyll serve --incremental
```

Clean:
```
bundle exec jekyll clean
```

Update with the latest PortalsJS runtime:
* Replace the `main.js` file with the new runtime after minifying and obfuscating it.
